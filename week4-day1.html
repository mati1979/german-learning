Imperative Forms (A1 core)

German has three main imperative “targets”: du (informal singular), ihr (informal plural), and Sie (formal). You’ll use all three — but du and Sie are the most important early.

1) du-imperative (informal singular)

Basic idea: use the verb stem (often like “du” form without -st). Sometimes an extra -e appears (less important at A1; optional in many cases).

Infinitivedu-formImperative (du)Meaning
kommendu kommstKomm!Come!
wartendu wartestWarte!Wait!
machendu machstMach!Do!/Make!
sagendu sagstSag!Say!/Tell!
hörendu hörstHör (zu)!Listen!
lesendu liestLies!Read!
nehmendu nimmstNimm!Take!
gebendu gibstGib!Give!
A1 rule of thumb: If it’s a regular verb, try “remove -st” from the du form: du machst → Mach!, du wartest → Warte!. For common irregulars, just memorize the few high-frequency ones (nimm, gib, lies).

2) ihr-imperative (informal plural)

Very easy: it looks like the ihr form without the pronoun.

Ihr kommt → Kommt!
Ihr wartet → Wartet!
Ihr macht → Macht!
Ihr hört zu → Hört zu!

3) Sie-imperative (formal)

This is the most polite and super useful in real life. Structure: Verb + Sie (and usually bitte).

Kommen Sie bitte!
Warten Sie bitte!
Machen Sie bitte die Tür zu.
Sagen Sie bitte Ihren Namen.
Important: With Sie, you keep the verb conjugation like normal present: Sie machen → Machen Sie!.

Negating imperatives (Don’t…)

Use nicht to negate an action, or kein to negate a noun. In imperatives, nicht typically comes after the object/phrase.

Mach das nicht! (Don’t do that!)
Komm nicht! (Don’t come!)
Ruf mich nicht an! (Don’t call me!)
Kauf kein Brot! (Don’t buy bread.)
Kauf keine Bücher! (Don’t buy books.)
A1 memory: “no/not any noun” = kein. “not (verb/phrase)” = nicht.

Daily Language: Commands, Requests, and Softening

Real German isn’t just “Do this!” It’s also: “Please…”, “Just a moment…”, “Could you…”. Today you’ll build a command bank you can reuse forever.

Polite softeners (A1)

PhraseMeaningUse
bitteplease / you’re welcomesoftens commands
maljust / for a momentcasual softener
einfachjust / simplyreduces pressure
kurzbrieflyquick action request
einen Momenta moment“wait a second”
Komm bitte.
Warte mal.
Hör bitte kurz zu.
Einen Moment bitte!
Note: mal is extremely common in spoken German. It makes commands sound less harsh.

Super high-frequency imperatives (learn these as chunks)

At home / daily

Komm! / Kommt! / Kommen Sie!
Warte! / Wartet! / Warten Sie!
Mach die Tür auf!
Mach die Tür zu!
Steh auf! (get up!)
Setz dich! (sit down!)
Komm mit! (come along!)
Sei bitte leise! (be quiet!)

Communication

Hör zu! (listen!)
Sag das nochmal. (say that again)
Sprich langsam. (speak slowly)
Schreib das auf. (write that down)
Lies das. (read that)
Gib mir das. (give me that)
Nimm das. (take that)
A1 trick: Build “command frames”: Mach ___ auf/zu, Sag ___, Gib mir ___. Then you can swap in vocabulary forever.

Drills (90 minutes) — automation through repetition

These drills are designed to make imperatives automatic and to integrate them with A1 skills: objects (accusative), separable verbs, negation, and polite language.

Block A (25 min) — Form building: du / ihr / Sie (45 forms)

For each verb, write 3 imperatives: du, ihr, Sie. Add bitte to the Sie form.

Verbs: kommen, warten, machen, sagen, hören, lesen, nehmen, geben, kaufen, trinken, essen, lernen, suchen, aufstehen, anrufen, einkaufen
Sample for 3 verbs
kommen: Komm! · Kommt! · Kommen Sie bitte!
warten: Warte! · Wartet! · Warten Sie bitte!
anrufen: Ruf an! · Ruft an! · Rufen Sie bitte an!
Don’t get stuck: if a du-form feels uncertain (irregular), you can still master Sie and ihr reliably. But for the high-frequency irregulars (nimm/gib/lies), memorize.

Block B (25 min) — Object frames (40 commands)

Write 40 commands using object frames. Mix: accusative objects, possessives, plural. Use both du and Sie.

Nouns: das Handy, das Buch, das Brot, die Tasche, die Tür, die Wohnung, die Bücher, die Taschen
Possessives: mein/dein/sein/ihr/unser
Example set (10)
Gib mir mein Handy.
Nimm dein Buch.
Mach die Tür zu.
Mach bitte die Tür auf.
Kaufen Sie bitte das Brot.
Lies mein Buch.
Such deine Tasche.
Rufen Sie bitte Ihren Freund an.
Kauf die Bücher nicht!
Mach das nicht!

Block C (20 min) — Negation: nicht vs kein (24 commands)

Write 24 negative imperatives: 12 with nicht, 12 with kein. Mix du and Sie.

Examples (not the full set)
Mach das nicht!
Ruf mich nicht an!
Komm nicht mit!
Kauf kein Brot!
Kauf keine Bücher!
Essen Sie bitte keine Pizza. (A1 structure)
Check: If you can replace it with “no/not any + noun” → use kein. Otherwise, use nicht.

Reading (25 minutes) — instructions in context

Read once out loud, then answer questions in simple A1 German. Imperatives are common in signs, messages, and instructions.

Hallo! Bitte komm kurz. Warte einen Moment.
Mach die Tür zu und setz dich.
Hör bitte zu: Schreib das auf.
Ruf deinen Freund an, aber ruf mich nicht an.
Kauf bitte Brot und zwei Brötchen. Kauf keinen Kaffee.

Tasks

  1. List 6 imperatives from the text.
  2. Which lines use nicht? Which use kein?
  3. Rewrite 3 lines in Sie-form (formal).
  4. Rewrite 2 lines in ihr-form (plural).
  5. Write 5 new imperatives using your own nouns (or from the toolbox).
Possible Sie-form rewrites
Bitte kommen Sie kurz. Warten Sie einen Moment.
Machen Sie bitte die Tür zu und setzen Sie sich.
Hören Sie bitte zu: Schreiben Sie das auf.

Speaking (20 minutes) — real-life roleplay

Imperatives are spoken language. This block makes them automatic. Use a timer. Speak out loud.

Part 1 (8 min) — command bursts

Do 8 rounds. Each round: pick 1 verb and give 4 commands quickly (du + Sie + negative + with object). Example verb: machen.

Example burst
Mach die Tür zu!
Machen Sie bitte die Tür zu!
Mach das nicht!
Mach bitte die Tür auf!

Part 2 (7 min) — mini situations

Speak 5 mini scenes (30–45 seconds each):

  1. Teacher / student: “Listen, repeat, write it down.”
  2. At home: “Open the door, close the door, wait.”
  3. At a café: “Please sit, wait a moment, don’t call.”
  4. Shopping: “Buy bread, don’t buy coffee.”
  5. Lost item: “Search my phone, give it to me.”

Part 3 (5 min) — polite vs direct

Say each command twice: once direct (du), once polite (Sie + bitte).

Komm. → Kommen Sie bitte.
Warte. → Warten Sie bitte.
Mach die Tür zu. → Machen Sie bitte die Tür zu.
Speaking rule: If you say the wrong form, repeat the corrected form twice. That’s the training.

Production (20 minutes) — make it yours (still A1)

Production converts drills into real language you can use.

Production 1 — Your personal command list (25 commands)

Write 25 commands you might actually use (teacher, daily life, travel). Requirements:

  • 10 du-imperatives
  • 10 Sie-imperatives (with bitte)
  • 5 negative imperatives (mix nicht/kein)

Production 2 — Message: “What to do” (8–10 lines)

Write a short message to a friend (informal) giving instructions for today. Include: 6 imperatives, 2 time words, 2 negatives, 1 plural noun.

Starter frame
Hey! Komm bitte um zwei Uhr.
Warte kurz. Mach die Tür zu.
Kauf Brot, aber kauf keinen Kaffee.
Ruf mich nicht an. Schreib mir!

Production 3 — Formal situation: receptionist / office (8–10 lines)

Write a formal instruction exchange using Sie. Include: 6 Sie-imperatives, 2 objects, 2 negatives.

Tomorrow (Day 2): Telling time. We’ll reuse imperatives to make time language automatic: “Come at 2”, “Wait until 3”, etc.

Exit Check (10 minutes)

Finish with a quick test: can you use imperatives without thinking too hard?

  • Write 10 du-imperatives (mix regular + separable verbs).
  • Write 10 Sie-imperatives with bitte.
  • Write 8 negative imperatives: 4 with nicht, 4 with kein.
  • Write 6 commands with objects + possessives (e.g., Gib mir mein Handy).
  • Speak 12 commands out loud (alternate du ↔ Sie). If you hesitate, repeat that verb set tomorrow.

Bug list

  1. The imperative form I hesitate on most is: du / ihr / Sie.
  2. My most common error is: (missing prefix / wrong negation / wrong object phrase).
  3. My fix tomorrow: 10 minutes of Block ___ + speaking bursts.
A1 truth: If you can give simple instructions, you can survive daily German. That’s the point of imperatives.
