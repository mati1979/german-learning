Level Contents (specific)

This is what each level contains. The hubs break it down week‑by‑week (starting with A1).

A1 · Beginner

Goal: stable main sentences + everyday basics.

  • Verb position (V2), sentence skeleton
  • sein/haben, regular verbs, basic questions
  • Nominative + Accusative, nicht vs kein
  • Modals, separable verbs, possessives
  • Time/date basics, short texts + writing

Open A1 Hub → · A1 weekly overview page →

A2 · Elementary

Goal: widen grammar and narrate past events.

  • Dative case + dative prepositions
  • Two‑way prepositions (location vs direction)
  • Perfekt (haben/sein), basic Präteritum (sein/haben/modals)
  • Reflexive verbs, comparisons
  • Subordinate clauses: weil / dass / wenn
  • Adjective endings (functional, not perfect)

Open A2 Hub →

B1 · Intermediate

Goal: independence + longer, accurate speech/writing.

  • All four cases solidified (Genitive basics included)
  • Präteritum widely, Plusquamperfekt basics, future
  • Passive voice, Konjunktiv II (würde)
  • Relative clauses, infinitive clauses (zu)
  • Opinions, reasons, narratives, formal emails

Open B1 Hub →

B2 · Upper‑Intermediate

Goal: nuance, register, and professional German.

  • Konjunktiv I (indirect speech) + deeper Konjunktiv II
  • Nominalization, participle constructions
  • Complex clause architecture, connectors
  • Academic/professional writing and argumentation
  • Workplace + bureaucracy vocabulary (PiA‑relevant)

How you add new days (minimal editing)

  1. Create a new file like week1-day2.html in the same folder.
  2. Open a1.html and add one link line under the right week.
  3. Done. Your homepage doesn’t change.

Static sites can’t “auto-detect” new files without JS/build tools. This hub structure keeps editing small and predictable.
