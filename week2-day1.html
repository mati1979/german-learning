Accusative Basics: Direct Object (wen/was?)

Week 1 gave you the sentence engine (V2). Week 2 upgrades meaning: now your verbs can hit something. Today you install the concept that unlocks real German: the direct object (Akkusativ). German doesn’t “love cases.” It loves clarity.

Today’s Overview

Your mission: write stable sentences with a direct object, without breaking V2. If Week 1 was “I am / I have,” Week 2 begins “I buy / I drink / I read … something.”

Mini-Lesson

1) The new piece: direct object

A direct object is what receives the action. English shows this mostly by position (“I buy coffee”). German also marks it with the accusative (Akkusativ) — mainly visible in the articles.

Core pattern Subject – Verb – Object

Ich kaufe den Kaffee.

Wir lesen das Buch.

Du trinkst das Wasser.

How to find the object: ask wen? (who/whom) or was? (what).
Example: Ich kaufe … was? → den Kaffee.

2) The big visible change: accusative articles

Good news: accusative is not a giant monster today. For most nouns in A1, only one article really changes: der → den (masculine). The others stay the same.

GenderNominative (Subject)Accusative (Direct object)Example
masculine der Mann den Mann Der Mann kauft den Kaffee.
feminine die Frau die Frau Die Frau kauft die Pizza.
neuter das Kind das Kind Ich sehe das Kind. (“see” comes later; pattern only)
plural die Kinder die Kinder Wir kaufen die Äpfel.
Today’s memory hack: In A1, accusative is basically: der → den and everything else mostly stays the same (for articles). Your job is not perfection. Your job is automatic recognition.

3) Indefinite articles (a/an): ein → einen (masculine only)

Same story. Only masculine changes. You will drill this until your brain stops negotiating.

GenderNominativeAccusativeExample
masculine ein Kaffee einen Kaffee Ich kaufe einen Kaffee.
feminine eine Pizza eine Pizza Du kaufst eine Pizza.
neuter ein Wasser ein Wasser Wir trinken ein Wasser. (restaurant German)

4) V2 still rules

You’re adding an object, but V2 stays the boss. Time-first sentences still keep the verb in position 2.

Pattern A Subject first

Ich kaufe den Kaffee.

Wir lesen das Buch.

Pattern B Time first → verb still #2

Heute kaufe ich den Kaffee.

Jetzt trinken wir das Wasser.

Abends essen wir eine Pizza.

Common failure mode: you focus on den and accidentally break V2. That’s illegal. V2 first, always.

Drills (writing + saying aloud)

The drills are in the only order that works: identify object → choose article → keep V2 stable → produce. Say every sentence aloud after writing it.

How to drill: Write → speak → underline the verb → underline the object → check the article. If wrong: rewrite correctly three times.

Drill Block A (Object Finder): “What is the object?”

Write 25 sentences. Then underline the direct object (the thing receiving the action). Don’t worry about perfect endings — worry about object awareness.

Model: Ich kaufe den Kaffee.

Model: Wir lesen das Buch.

Underline: den Kaffee / das Buch

  • Use only these verbs: kaufen, essen, trinken, lesen
  • Use only these objects: den Kaffee, den Apfel, das Wasser, das Brot, das Buch, die Pizza, die Banane
  • Use 5+ different subjects (ich/du/er/sie/wir/ihr/sie)
Sample answers (peek only if needed)

Ich kaufe den Kaffee. — Du isst die Pizza. — Er trinkt das Wasser. — Wir lesen das Buch. — Ihr kauft den Apfel.

Drill Block B (Article Training): “der → den” until boring

Write 30 sentences that contain masculine objects. Your only job is to write den correctly and not sabotage V2.

Model: Ich kaufe den Kaffee.

Model: Wir essen den Apfel.

  • Use masculine objects only: den Kaffee / den Apfel
  • Mix verbs: kaufen, essen, haben
  • 10 sentences must be time-first (V2): Heute / Jetzt / Morgens / Abends / Oft
Mini checklist
  • I wrote den 20+ times
  • I did 10+ time-first sentences with correct V2
  • I did not accidentally write der Kaffee as an object

Drill Block C (Mix + Speed): “V2 + object + variety”

Write 40 mixed sentences. Half subject-first, half time-first. This is where your brain tries to complain. Ignore it.

  • Include 15+ masculine objects (den Kaffee / den Apfel)
  • Include 10+ neuter objects (das Wasser / das Brot / das Buch)
  • Include 10+ feminine objects (die Pizza / die Banane)
  • Use all 5 verbs at least once
Speed target

If you can write 40 correct-ish sentences in ~25 minutes while keeping V2 stable, you’re on track for A1 mastery. If you can’t yet, that’s normal — you repeat until you can.

Quizzes (mean but fair)

These force accuracy. Do them fast, then check answers. If you miss a pattern, you go drill that pattern again. No drama.

Quiz 1 — Identify subject vs object

Mark S (subject) and O (object) for the underlined part.

  1. Ich kaufe den Kaffee.
  2. Der Mann kauft den Apfel.
  3. Wir lesen das Buch.
  4. Die Frau trinkt das Wasser.
  5. Ihr esst die Pizza.
Answers
  1. O
  2. S
  3. O
  4. S
  5. O

  4. Ihr lest ____ Buch.

  2. den

  4. das

Production (prove you own it)

This is where you stop being a student and start being someone who can produce German.

Production 1 — “Accusative Ladder” (12 sentences)

Write 12 sentences. Each step adds control. Keep V2 correct.

  1. Ich kaufe den Kaffee.
  2. Time-first: Heute kaufe ich den Kaffee.
  3. Change object: Ich kaufe den Apfel.
  4. Change verb: Ich esse den Apfel.
  5. Change gender (feminine): Ich kaufe die Pizza.
  6. Neuter: Ich trinke das Wasser.
  7. Plural (optional): Wir kaufen die Äpfel.
  8. Use du: Du liest das Buch.
  9. Time-first with du: Abends liest du das Buch.
  10. Use wir: Wir essen eine Pizza.
  11. Indefinite masculine: Ich kaufe einen Apfel.
  12. Finish: a mixed sentence you invent (must include V2 + object).
Strict rule

If you hesitate: write simpler, not fewer. The win is stable structure + correct object marking (especially der → den).

Production 2 — Mini paragraph (6–8 sentences)

No dictionary. Use only today’s verbs and objects + Week 1 time words.

Write about:

  • What you buy/eat/drink/read today
  • What you do in the morning / evening

Rules: include 3 time-first sentences (V2!) and 3 masculine objects with den.

Example (do not copy — just compare)

Hallo! Heute kaufe ich den Kaffee. Jetzt trinke ich das Wasser. Morgens esse ich den Apfel. Abends lese ich das Buch. Oft kaufe ich den Apfel und eine Pizza.

Real A1 skill: you can produce without freezing. Articles can be corrected later — but only if your sentence structure exists.
